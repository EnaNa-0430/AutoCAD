# 位图图像最小参数曲线拟合方案

## 一、问题定义

目标：
将位图图像（PNG/JPG）中的闭合曲线转换为**最小参数的几何表达**，例如：

- 圆 → 使用 (x, y, r)
- 直线 → 使用两点或参数式
- 椭圆 → 使用 5 个参数

而不是使用大量折线逼近。

最终输出：
- 参数化结构（JSON）
- 矢量图（SVG）
- 与原图对比可视化

---

## 二、整体流程（Pipeline）

```
输入图像
   ↓
1. 预处理（二值化 / 去噪）
   ↓
2. 边缘提取 + 轮廓检测
   ↓
3. 轮廓分段（曲线分解）
   ↓
4. 基元识别 + 参数拟合
   ↓
5. 模型选择（最小参数原则）
   ↓
6. 矢量化输出（SVG / CAD）
```

---

## 三、模块设计

### 1. 图像预处理

目标：获得干净的轮廓

方法：
- 灰度化
- 自适应阈值
- 形态学操作（开运算 / 闭运算）

关键函数：
```python
cv2.threshold
cv2.morphologyEx
```

---

### 2. 边缘检测 + 轮廓提取

方法：
- Canny 边缘检测
- findContours

```python
edges = cv2.Canny(img, t1, t2)
contours, _ = cv2.findContours(edges, ...)
```

输出：点集

---

### 3. 轮廓分段

必要性：不同曲线类型需要分开拟合

方法：

#### 方法1：曲率分析
- 计算离散曲率
- 检测突变点

#### 方法2：RDP算法
- 根据误差进行简化

---

### 4. 基元拟合

候选模型：

| 类型 | 参数数量 |
|------|--------|
| 直线 | 2 |
| 圆 | 3 |
| 椭圆 | 5 |
| 圆弧 | 4 |
| 二次曲线 | 6 |
| B样条 | 多 |

---

### 4.1 拟合方法

- 直线：最小二乘
- 圆：Kasa / Pratt
- 椭圆：Fitzgibbon

---

### 4.2 RANSAC（抗噪）

流程：
1. 随机采样
2. 拟合模型
3. 计算内点
4. 选择最优模型

---

### 5. 模型选择（核心）

目标：用最少参数表达

方法：

#### AIC / BIC

```
Score = Error + λ * Complexity
```

说明：
- Error：拟合误差
- Complexity：参数数量

---

### 6. 闭合曲线判断

```
||P0 - Pn|| < ε
```

若闭合：优先尝试
- 圆
- 椭圆

---

### 7. 参数化输出

```json
[
  {
    "type": "circle",
    "center": [x, y],
    "radius": r
  },
  {
    "type": "line",
    "p1": [...],
    "p2": [...]
  }
]
```

---

### 8. SVG输出

```xml
<circle cx="100" cy="100" r="50" />
<line x1="0" y1="0" x2="100" y2="100" />
```

### 9. 可视化对比

- 拟合结果（红）
